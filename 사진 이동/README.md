# 웹 사이트 제작 실습

## Header 부분
    1. 왼쪽에 로고를 띄운다(임의의 이미지와 문구)
    2. 오른쪽에 Navigation menu 들을 위치한다
      > 마우스를 올리면 submenu가 내려온다
    3. contact를 누르면 contact를 작상하는 form이 뜬다.
     
     
## 중앙 슬라이드 부분
    1. 사진 정중앙에 임의의 문구를 입력
    2. 사진 중앙 상단에 동그란 버튼을 클리하면 다른 사진으로 슬라이드 이동
      > 임의의 사진 사용
    
## 콘텐츠 부분
    중앙 슬라이드 하단 좌측에 공지사항 부분이다
      1. 테이블에는 내용과 날짜가 있다
      2. 테이블의 행은 교차로 색깔이 다르다.
      3. 클릭은 가능하나 링크를 연결하지 않았다.
    
    
    중앙 슬라이드 하단 우측에 Partner Site 부분
      1. 문구 밑에 임의의 사진과 임의의 문구로 이루어진 사진이 있다
      2. 사진 클릭시 팝업창으로 새로운 사진이 뜬다.
      3. 닫기를 누르면 사라진다

## 아이콘
    임의의 아이콘 3개로 이루어져 있고, 각자 임의의 버튼 3개를 만들었다
    아무 기능이 없는 아이콘들이다
    
    
## footer 부분
    법적 고지, 개인정보 최급 방침 등의 문구
    오른쪽에 임의의 사이트 정보가 들어있다.
